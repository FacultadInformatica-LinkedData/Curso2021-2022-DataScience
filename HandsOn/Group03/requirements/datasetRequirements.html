(R1) Data are in one of the selected smart city domains

Data set showing how much fuel each municipality vehicle has consumed, by year and by fuel type in the UK.

(R2)Data are available as a CSV file

Data set is accessible in csv format.

(R3)Data have an open license, so they can be published

The dataset is openly licensed by the UK government, so that the information can be used, manipulated and published.

(R4) Data can be easily linked with generic real-world entities (e.g., locations).

The data set is linked and has the following cells:

  • Fleet number (specific number used to identify a vehicle)
  • Vehicle (brand and model)
  • Fuel used in liters
  • Total mileage for the reporting period
  • MPG = Miles per gallon
  • Type of fuel
