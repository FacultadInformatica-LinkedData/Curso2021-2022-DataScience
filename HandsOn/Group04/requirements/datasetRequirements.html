Grupo 4

El dataset seleccionado para el proyecto recoge informacion variada sobre contenedores de reciclaje inteligentes instalados en el municipio de Las Rozas

    Lista de Requirimientos de los Datos

  1. Los datos deben de estar relacionados con las Smart Cities
  2. Los datos deben estar en formato CSV
  3. Los datos deben tener una licencia abirta para ser publicados y usados
  4. Los datos deben de ser fácilmente conectables a entidades genericas y reales

Relación de los datos con las Smart Cities

Las Rozas es un municipio que apuesta por la innovacion y la eficiencia con proyectos que la llevan a ser una smart city.

Datos en formato CSV

El dataset elegido esta en formato .csv

Los datos tienen una licencia abierta

Dado que el ayuntamiento de Las Rozas esta volcada en la innovacion sus datos tienen licencia abierta tanto para uso comercial como no comercial.

Los datos son datos reales proporcionados directamente por el ayuntamiento ante un problema real que tienen.

Los datos son de lugares, direcciones, fechas, tipos... Todos ellos reales.
